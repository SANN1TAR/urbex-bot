URBEX-BOT — Документация v0.1
==============================

ФАЙЛЫ:
------

bot.py
  Главный файл. Запускает Telegram-бота, обрабатывает все команды и кнопки.
  Зависит от: database.py, search.py
  Запуск: python bot.py

database.py
  Работа с SQLite базой данных.
  Хранит пользователей: telegram_id, имя, город.
  Функции: init_db(), get_user(id), save_user(id, name, city)

search.py
  Поиск объектов через DuckDuckGo + обработка через Gemini AI.
  Функции:
    search_objects(type, city) — ищет 3 объекта по типу и городу
    search_images(name, city) — ищет 2 фото объекта
    search_by_name(name, city) — ищет конкретный объект по названию

requirements.txt
  Все зависимости Python.

.env
  Секретные ключи: TELEGRAM_TOKEN, GEMINI_API_KEY
  НЕ КОММИТИТЬ в git.

urbex_bot.db
  База данных SQLite. Создаётся автоматически при первом запуске.

КНОПКИ БОТА:
------------
🏚️ Заброшка — ищет 3 заброшенных здания в городе пользователя
🏗️ Крыша — ищет 3 крыши/высотки для руфинга
⛏️ Бомбарь — ищет 3 бомбоубежища/подземелья
🔍 Поиск по названию — ищет конкретный объект по введённому названию

РЕГИСТРАЦИЯ:
-----------
При первом /start бот спрашивает имя и город.
Данные сохраняются в БД и используются при каждом поиске.
