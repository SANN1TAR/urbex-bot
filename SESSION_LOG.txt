SESSION_LOG — Urbex Bot
=======================

[2026-04-30] Сессия 1
----------------------
СДЕЛАНО:
- Создан проект urbex-bot с нуля
- Реализована регистрация (имя + город) через FSM aiogram 3
- 4 кнопки: Заброшка, Крыша, Бомбарь, Поиск по названию
- Поиск через DuckDuckGo (бесплатно, без API)
- Обработка результатов через Google Gemini 1.5 Flash (бесплатный tier)
- Выдача по 3 объекта с названием, адресом, описанием, фото + источник фото
- Поиск по конкретному названию объекта
- SQLite база данных для хранения пользователей
- Все ключи в .env файле

СТЕК:
- Python + aiogram 3.7
- Google Gemini 1.5 Flash API
- DuckDuckGo Search
- aiosqlite

КЛЮЧИ:
- TELEGRAM_TOKEN — токен бота от @BotFather
- GEMINI_API_KEY — ключ Google AI Studio (бесплатный)

ЧТО ДАЛЬШЕ:
- Установить зависимости: pip install -r requirements.txt
- Запустить: python bot.py
- Протестировать все 4 кнопки
- Возможные улучшения: кэш результатов, кнопка "ещё 3", смена города
