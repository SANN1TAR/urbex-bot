VERSIONS — Urbex Bot
====================

v0.1 [2026-04-30]
  - Первая версия: регистрация имя+город, 4 кнопки, поиск через DuckDuckGo+Gemini

v0.2 [2026-04-30]
  - Замена Gemini на Groq, DuckDuckGo на Tavily
  - Убраны диггеры/бомбари

v0.3 [2026-04-30]
  - Тиндер-интерфейс: по одному объекту, кнопки Следующий/Заново
  - Иерархия источников: Викимапия→Урбантрип→Telegram→VK→общий
  - /restart, фильтр запрещённых объектов, координаты только из источника

v0.4 [2026-05-01]
  - Удалён руфинг/крыши
  - Архитектура: 1 вызов LLM вместо 7 за запрос
  - Расширены фильтры BANNED_WORDS и JUNK_RE
  - Дедупликация объектов по имени и адресу
  - Убрана фраза "место не помечено на карте"

v0.5 [2026-05-01]
  - Полный переход на urban3p.ru как источник данных
  - SQLite кеш объектов с TTL 7 дней + фоновое обновление
  - Парсинг страниц urban3p.ru: OG фото + /onmap координаты
  - Nominatim как fallback для координат
  - Карточка объекта: только название + локация + фото (без описания)
  - Команда /help с полной справкой
  - LLM используется только для поиска по названию (/поиск)
